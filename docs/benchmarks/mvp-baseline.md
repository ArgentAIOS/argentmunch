signatures and summaries. Queries return only relevant symbols — typically 
90-99%+ token reduction.

### Key Takeaways

1. Even broad queries (`index`) return dramatically fewer tokens than raw file dumps
2. Specific lookups (`save_index`) achieve near-perfect efficiency
3. The one-time indexing cost amortizes over all future queries
